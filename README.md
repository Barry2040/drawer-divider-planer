# 📐 The Drawer Dividers - 3D Web Planner

Willkommen zum interaktiven 3D Web-Planer für das **Drawer Dividers V2** System! 🚀

Dieses Open-Source-Tool hilft dir dabei, deine Schubladen virtuell zu planen, bevor du den 3D-Drucker anwirfst. Keine Zettel, keine komplizierte Mathematik und kein lästiges Skalieren im Slicer mehr!

👉 **[[Hier geht es direkt zum Web-Planer]](https://barry2040.github.io/drawer-divider-planer/)**

## ✨ Features
* **Interaktive 3D-Vorschau:** Betrachte deine Schublade live in 3D (angetrieben durch Three.js).
* **Millimetergenau:** Gib einfach die Innenmaße deiner echten Schublade ein. Das Tool generiert den passenden virtuellen Raum.
* **Visuelles Layouting:** Füge Trennwände per Knopfdruck hinzu und schiebe sie stufenlos an die perfekte Position.
* **Auto-Kalkulation:** Das Tool berechnet automatisch, welche Länge deine zu druckenden Trennwände haben müssen.

## 🛠️ Wie nutze ich den Planer?
1. **Maße eingeben:** Miss die Breite und Tiefe deiner Schublade und trage die Werte in die Schieberegler ein.
2. **Wände platzieren:** Klicke auf "Wand hinzufügen" und schiebe sie an die gewünschte Stelle.
3. **Werte ablesen:** Notiere dir die Positionen und berechneten Längen.
4. **Drucken:** Gehe rüber zu MakerWorld und trage die Längen in meine parametrischen Generatoren ein. Fertig!

## 💻 Für Entwickler (Lokale Installation)
Dieses Projekt ist komplett webbasiert (HTML, CSS, JavaScript) und nutzt die Three.js Bibliothek via CDN. Es ist kein Build-Prozess oder npm-Install nötig!
1. Repository klonen: `git clone https://github.com/DEIN_NAME/drawer-divider-planer.git`
2. Die Datei `index.html` über einen lokalen Webserver (z. B. VS Code "Live Server") öffnen.
3. Code anpassen und Spaß haben!
